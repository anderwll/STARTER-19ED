Bootstrap - Grid

O Bootstrap é um framework front-end que facilita a criação de sites responsivos. Ele é composto por um conjunto de classes CSS e JavaScript que ajudam a estruturar o layout de uma página web.

Basta incluir o arquivo CSS do Bootstrap no cabeçalho do documento HTML e começar a utilizar as classes fornecidas pelo framework.

Também é possível utilizar o Bootstrap através de um CDN (Content Delivery Network), que é um serviço que armazena arquivos em servidores distribuídos geograficamente, permitindo um carregamento mais rápido dos recursos. Dessa forma, não é necessário baixar os arquivos do Bootstrap para o seu computador, basta incluir os links para os arquivos no seu documento HTML.
